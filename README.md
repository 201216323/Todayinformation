# Todayinformation
最新鲜的时事新闻，最时尚的手机科技，最好玩的热门游戏，还有各式各样的微博热点。 第一时间听到今日的最新资讯，24小时和你分享生活的每一分钟！

这是本人毕业设计所做的一个小项目，主要是模仿今日头条这样的资讯平台的。其中借鉴了一些开源框架和大神博客的代码，本着开源的精神，如有转载，请注明出处。下面是项目的一些贴图。
